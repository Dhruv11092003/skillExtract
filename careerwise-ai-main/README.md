# careerwise-ai
AI-powered resume ranker and career path suggestor using NLP and machine learning.

Careerwise is an intelligent system that helps job seekers and students by:

Ranking resumes based on their relevance to a given job description.

Suggesting career paths aligned with the user's current skills, experience, and interests.

Utilizing Natural Language Processing (NLP) and Machine Learning (ML) techniques for accurate analysis and recommendations.
